What is a Variable?

A named box in memory that stores a value.

Primitive Data Types

Java has 8 primitive types. The 5 most common for beginners:

int age = 20; // whole numbers / números enteros double gpa = 3.75; // decimals / decimales String name = "Maria"; // text / texto (NOT primitive) boolean passed = true; // true or false / verdadero o falso char grade = 'A'; // single character / carácter único
💡

Naming rule: variable names must start with a letter, can't have spaces, and Java is case-sensitive (age ≠ Age).

Declaring & Assigning

// Declare and assign in one line / Declarar y asignar en una línea int score = 95; // Declare first, assign later / Declarar primero, asignar después int bonus; bonus = 10; // Update an existing variable / Actualizar variable existente score = score + bonus; // score is now 105 System.out.println(score); // prints 105

Arithmetic Operators

Math inside your code.

int a = 10, b = 3; System.out.println(a + b); // 13 → addition / suma System.out.println(a - b); // 7 → subtraction / resta System.out.println(a * b); // 30 → multiplication / multiplicación System.out.println(a / b); // 3 → integer division / división entera System.out.println(a % b); // 1 → modulus/remainder / módulo/residuo
⚠️

Integer division: 10/3 = 3 (not 3.33). Java drops the decimal when both numbers are int.

Comparison Operators

int x = 5, y = 8; System.out.println(x == y); // false → equal / igual System.out.println(x != y); // true → not equal / diferente System.out.println(x < y); // true → less than / menor que System.out.println(x >= y); // false → greater or equal / mayor o igual

Logical Operators

boolean hasTicket = true; boolean isAdult = false; System.out.println(hasTicket && isAdult); // false (AND — both must be true) System.out.println(hasTicket || isAdult); // true (OR — at least one true) System.out.println(!hasTicket); // false (NOT — flips the value)

The backbone of decision-making in Java.

Module 6: Methods & Functions

Organize reusable code into named blocks — the foundation of clean programming.

🔧

Defining & Calling Methods

Write once, use anywhere in your class.

// Method with no return value / Método sin valor de retorno static void greet(String name) { System.out.println("Hello, " + name + "!"); } // Method that returns a value / Método que retorna un valor static int add(int a, int b) { return a + b; } // Calling methods / Llamar métodos greet("Maria"); // Hello, Maria! int result = add(5, 3); // result = 8

Anatomy of a Method

  1. Return type: void (nothing) or a data type like int, double, String.
  2. Method name: descriptive camelCase name.
  3. Parameters: input values inside ().
  4. Body: code in {} that executes.
  5. return: sends a value back (not needed for void).

Module 8: Inheritance

Build new classes on top of existing ones — reuse, extend, and override.

extendssuper@Overridepolymorphism
🧬

Parent & Child Classes

A child class inherits fields and methods from its parent.

The extends Keyword

// Parent class / Clase padre class Animal { String name; Animal(String n) { name = n; } void speak() { System.out.println(name + " makes a sound"); } } // Child class inherits from Animal / Clase hija hereda de Animal class Dog extends Animal { Dog(String n) { super(n); } // calls parent constructor @Override void speak() { System.out.println(name + " says: Woof!"); } } // Usage / Uso Animal a = new Animal("Generic"); Dog d = new Dog("Rex"); a.speak(); // Generic makes a sound d.speak(); // Rex says: Woof!
💡

super() calls the parent's constructor. Must be the first line in the child constructor.

Polymorphism

A parent-type variable can hold any child object.

Animal[] zoo = { new Animal("Cat"), new Dog("Rex") }; for (Animal x : zoo) { x.speak(); // calls the correct speak() for each object! } // Output: Cat makes a sound Rex says: Woof!

🚗 Vehicle Hierarchy

class Vehicle { int speed; Vehicle(int s) { speed = s; } void move() { System.out.println("Moving at " + speed + " mph"); } } class Car extends Vehicle { String brand; Car(String b, int s) { super(s); brand = b; } @Override void move() { System.out.println(brand + " cruising at " + speed + " mph"); } } Car c = new Car("Toyota", 65); c.move();
Toyota cruising at 65 mph

📐 Shape Area — super method call

class Shape { String color; Shape(String c) { color = c; } void describe() { System.out.println("Color: " + color); } } class Circle extends Shape { double radius; Circle(String c, double r) { super(c); radius = r; } @Override void describe() { super.describe(); // call parent method first System.out.println("Radius: " + radius); } } new Circle("Red", 5.0).describe();
Color: Red Radius: 5.0

🏦 Multi-level Inheritance

class Account { double balance; Account(double b) { balance = b; } void show() { System.out.println("Balance: $" + balance); } } class SavingsAccount extends Account { double rate; SavingsAccount(double b, double r) { super(b); rate = r; } void addInterest() { balance += balance * rate; } } SavingsAccount sa = new SavingsAccount(1000, 0.05); sa.addInterest(); sa.show(); // inherits show() from Account
Balance: $1050.0

Module 10: Exception Handling & File I/O

Handle errors gracefully and read/write files — write programs that survive the real world.

try-catchfinallythrowsFileWriterScanner
🛡️

Exception Handling

Catch runtime errors instead of crashing.

try-catch-finally

try { // code that might throw an exception int result = 10 / 0; // ArithmeticException! System.out.println(result); } catch (ArithmeticException e) { System.out.println("Error: " + e.getMessage()); // / by zero } finally { System.out.println("This always runs"); }
💡

try: code to attempt. catch: runs if exception occurs. finally: always runs (use for cleanup).

Common Exception Types

ArithmeticException // division by zero / división por cero NullPointerException // using a null reference / usar referencia null ArrayIndexOutOfBoundsException // bad array index NumberFormatException // "abc" parsed as int / parsear texto como int IOException // file read/write problems / problemas con archivos

File I/O — Writing & Reading

import java.io.*; // WRITING to a file / ESCRIBIR en un archivo try (FileWriter fw = new FileWriter("output.txt")) { fw.write("Hello, File World!\n"); fw.write("Second line"); } catch (IOException e) { System.out.println("Write error: " + e.getMessage()); } // READING from a file / LEER de un archivo try (Scanner sc = new Scanner(new File("output.txt"))) { while (sc.hasNextLine()) { System.out.println(sc.nextLine()); } } catch (FileNotFoundException e) { System.out.println("File not found!"); }

int[] arr = {1, 2, 3}; try { System.out.println(arr[10]); // index out of bounds } catch (ArrayIndexOutOfBoundsException e) { System.out.println("Bad index!"); } catch (Exception e) { System.out.println("General error: " + e.getMessage()); } finally { System.out.println("Done."); }

Java Swing — Core Components

Swing is Java's built-in GUI toolkit — no extra installs needed.

Your First Window

import javax.swing.*; // import all Swing classes public class HelloGUI { public static void main(String[] args) { // 1. Create window (JFrame) JFrame frame = new JFrame("My First App"); frame.setSize(400, 300); frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); // 2. Create components JLabel label = new JLabel("Hello, World!"); JButton btn = new JButton("Click Me"); // 3. Add to frame frame.setLayout(new FlowLayout()); frame.add(label); frame.add(btn); // 4. Make visible frame.setVisible(true); } }

Handling Button Clicks

JButton btn = new JButton("Click Me"); // Lambda-style ActionListener (Java 8+) btn.addActionListener(e -> { System.out.println("Button clicked!"); JOptionPane.showMessageDialog(null, "You clicked!"); });
💡

Key components: JFrame (window), JLabel (text), JButton (button), JTextField (input), JPanel (container).

GUI App Pattern

  1. Create JFrame and set size/close operation.
  2. Create components (JLabel, JButton, JTextField…).
  3. Add ActionListeners to interactive components.
  4. Set a layout manager (FlowLayout, BorderLayout, GridLayout).
  5. Add components to frame (or to a JPanel first).
  6. Call frame.setVisible(true) last.
