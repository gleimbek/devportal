Display & Flexbox

One-dimensional layouts made easy.

display Property

.block { display: block; } /* full width, new line */ .inline { display: inline; } /* width = content, same line */ .ib { display: inline-block; } /* inline + respects width/height */ .none { display: none; } /* hides completely */ .flex { display: flex; } /* flexbox container */ .grid { display: grid; } /* grid container */

Flexbox Essentials

.container { display: flex; flex-direction: row; /* row | column */ justify-content: space-between; /* main axis */ align-items: center; /* cross axis */ flex-wrap: wrap; /* allow wrapping */ gap: 16px; /* space between items */ } .item { flex: 1; /* grow to fill space equally */ flex-shrink: 0; /* don't shrink */ align-self: flex-start; /* override container alignment */ }

🔧 Common Flex Patterns

/* Centered hero */ .hero { display: flex; justify-content: center; align-items: center; min-height: 100vh; } /* Nav bar */ nav { display: flex; justify-content: space-between; align-items: center; padding: 0 24px; } /* Card row that wraps */ .cards { display: flex; flex-wrap: wrap; gap: 20px; } .card { flex: 1 1 280px; } /* grow, shrink, min 280px */

🔲 CSS Grid

.grid { display: grid; grid-template-columns: 1fr 2fr 1fr; /* 3 cols, middle is 2x */ grid-template-rows: auto 1fr auto; gap: 20px; } /* Responsive grid — auto-fill columns ≥ 200px */ .auto-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); gap: 16px; } /* Span across columns */ .header-area { grid-column: 1 / -1; } /* full width */ .sidebar { grid-row: 2 / 4; } /* span 2 rows */

📌 CSS Positioning

.static { position: static; } /* default */ .relative { position: relative; } /* offset from normal */ .absolute { position: absolute; } /* relative to nearest positioned ancestor */ .fixed { position: fixed; } /* relative to viewport */ .sticky { position: sticky; } /* sticks on scroll */ /* Centering with absolute */ .centered { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }
