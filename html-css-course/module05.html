Font Properties

Control how type looks on screen.

Core Typography Properties

body { font-family: 'Inter', Arial, sans-serif; /* font stack */ font-size: 16px; /* base size */ line-height: 1.6; /* 1.5–1.8 for readability */ color: #1e293b; /* dark but not pure black */ } h1 { font-size: clamp(1.8rem, 4vw, 3rem); /* fluid sizing */ font-weight: 700; /* 100–900 */ letter-spacing: -0.02em; /* tight for headings */ text-transform: none; /* uppercase | capitalize */ } p { max-width: 65ch; /* optimal line length */ text-align: left; /* left | center | right | justify */ text-indent: 1.5em; /* first line indent */ }
💡

65ch rule: Limit paragraph width to ~65 characters for maximum readability. The ch unit equals the width of "0" in the current font.

🔤 Using Google Fonts

<!-- 1. Add in <head> of HTML --> <link rel="preconnect" href="https://fonts.googleapis.com"> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap" rel="stylesheet"> /* 2. Use in CSS */ body { font-family: 'Inter', sans-serif; } /* Or via @import in CSS file */ @import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&display=swap'); h1 { font-family: 'Playfair Display', serif; }

🎨 Text Decoration & Effects

.link { text-decoration: underline; /* none | underline | line-through */ text-decoration-color: #3b82f6; text-underline-offset: 4px; /* gap between text and underline */ } .gradient-text { background: linear-gradient(135deg, #1d4ed8, #9333ea); -webkit-background-clip: text; -webkit-text-fill-color: transparent; } .truncate { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; /* shows "…" */ }

🎛️ CSS Custom Properties (Variables)

/* Define in :root — globally available */ :root { --font-sans: 'Inter', sans-serif; --font-serif: 'Playfair Display', serif; --text-base: 1rem; --text-lg: 1.25rem; --text-xl: 1.5rem; --color-text: #1e293b; } /* Use with var() */ body { font-family: var(--font-sans); color: var(--color-text); } h1 { font-family: var(--font-serif); font-size: var(--text-xl); }
