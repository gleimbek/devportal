The <video> Element

Native HTML5 video with full browser controls.

Video Element — Full Example

<!-- Basic video with controls --> <video width="800" height="450" controls poster="thumbnail.jpg" preload="metadata" > <!-- Multiple formats for browser compatibility --> <source src="video.webm" type="video/webm"> <source src="video.mp4" type="video/mp4"> <!-- Captions / subtitles --> <track kind="subtitles" src="captions-en.vtt" srclang="en" label="English" default > <track kind="subtitles" src="captions-es.vtt" srclang="es" label="Español"> <!-- Fallback for very old browsers --> <p>Your browser doesn't support HTML5 video. <a href="video.mp4">Download the video</a>. </p> </video>

Video Attributes Reference

controls — Show browser play/pause/volume controls autoplay — Play automatically (requires muted on most browsers) muted — Start without sound (needed for autoplay) loop — Repeat when it ends poster — Image shown before playback starts preload — none | metadata | auto playsinline — Play inline on iOS (not fullscreen) width/height — Prevents layout shift (like with images)
⚠️

Never autoplay with sound. Browsers block it by default. If you need autoplay (e.g. for background video), always add the muted attribute.

🔊 The <audio> Element

<audio controls preload="metadata"> <source src="podcast.ogg" type="audio/ogg"> <source src="podcast.mp3" type="audio/mpeg"> <p>Browser doesn't support audio. <a href="podcast.mp3">Download MP3</a></p> </audio> /* Audio formats: MP3 — Universal (most widely used) OGG — Open source, good quality WAV — Uncompressed, large file size AAC — High quality (Apple/iOS) WebM — Modern, excellent compression */

▶ Embedding YouTube / Vimeo

<!-- YouTube embed — use the /embed/ URL --> <iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" title="Video title for accessibility" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen loading="lazy" ></iframe> /* Responsive 16:9 iframe wrapper */ .video-wrap { position: relative; padding-bottom: 56.25%; /* 16:9 ratio */ height: 0; overflow: hidden; } .video-wrap iframe { position: absolute; top: 0; left: 0; width: 100%; height: 100%; }

🎨 Styling Video Elements

/* Always responsive */ video { max-width: 100%; height: auto; display: block; border-radius: 12px; } /* Background video hero */ .hero { position: relative; overflow: hidden; height: 100vh; } .hero video { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); min-width: 100%; min-height: 100%; object-fit: cover; } /* Overlay for readability */ .hero::after { content: ''; position: absolute; inset: 0; background: rgba(0,0,0,.5); }

📝 WebVTT Caption Format

/* captions-en.vtt */ WEBVTT 00:00:00.000 --> 00:00:03.500 Welcome to our HTML and CSS course. 00:00:03.500 --> 00:00:07.000 In this module, we'll learn about video and audio elements. 00:00:07.000 --> 00:00:11.000 <v Speaker>The <video> element gives you full control over media playback.
💡

Captions are required for accessibility under WCAG 2.1. They also benefit users in noisy environments or who prefer reading over listening.
