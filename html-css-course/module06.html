The Anchor Element

Everything you need to know about <a>.

Link Types & Attributes

<!-- External link — opens in new tab --> <a href="https://www.example.com" target="_blank" rel="noopener noreferrer"> Visit Example </a> <!-- Internal relative link --> <a href="about.html">About Us</a> <a href="../index.html">Up one folder</a> <a href="pages/contact.html">Down into folder</a> <!-- Jump to an ID on the page --> <a href="#section-2">Skip to Section 2</a> <section id="section-2">…</section> <!-- Email and phone links --> <a href="mailto:hello@example.com">Email Us</a> <a href="tel:+15551234567">Call Us</a> <!-- Download link --> <a href="resume.pdf" download>Download PDF</a>
⚠️

Always add rel="noopener noreferrer" to target="_blank" links — it prevents the new tab from accessing your page via window.opener (a security risk).

🎨 Styling Link States

/* LVHA order matters! */ a:link { color: #1d4ed8; } /* unvisited */ a:visited { color: #7c3aed; } /* visited */ a:hover { color: #2563eb; text-decoration: underline; } a:active { color: #1e40af; } /* being clicked */ a:focus-visible { outline: 3px solid #3b82f6; outline-offset: 3px; } /* Button-style link */ .btn-link { display: inline-block; padding: 10px 24px; background: #1d4ed8; color: #fff; border-radius: 8px; text-decoration: none; }

🧭 Navigation with Flexbox

<nav aria-label="Main navigation"> <ul> <li><a href="/" aria-current="page">Home</a></li> <li><a href="/about">About</a></li> <li><a href="/contact">Contact</a></li> </ul> </nav> /* CSS */ nav ul { display: flex; list-style: none; gap: 24px; } nav a { text-decoration: none; color: inherit; } nav a[aria-current] { font-weight: 700; border-bottom: 2px solid currentColor; }

♿ Link Accessibility

<!-- ❌ BAD — meaningless link text --> <a href="article.html">Click here</a> <!-- ✅ GOOD — descriptive text --> <a href="article.html">Read our guide to CSS Grid</a> <!-- ✅ Icon-only link needs aria-label --> <a href="https://twitter.com/x" aria-label="Follow us on Twitter"> 🐦 </a> <!-- Opens in new tab? Tell the user! --> <a href="https://example.com" target="_blank" aria-label="Example (opens in new tab)"> Example ↗ </a>
