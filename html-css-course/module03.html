How CSS Works

Rules, selectors, properties, and values.

CSS Rule Anatomy

/* selector { property: value; } */ h1 { color: #1d4ed8; font-size: 2rem; margin-bottom: 16px; } p, li { /* group selector */ line-height: 1.7; } .card { /* class selector */ background: #fff; border-radius: 8px; } #hero { /* ID selector (use sparingly) */ padding: 60px 24px; }

Three Ways to Add CSS

<!-- 1. External (best) — in <head> --> <link rel="stylesheet" href="styles.css"> <!-- 2. Internal — in <head> --> <style> body { background: #f0f4ff; } </style> <!-- 3. Inline (avoid for maintainability) --> <p style="color:red;">Red text</p>

🎯 CSS Selectors

* { box-sizing: border-box; } /* universal */ p { color: #333; } /* element */ .highlight { background: yellow; } /* class */ #logo { width: 120px; } /* ID */ a:hover { color: blue; } /* pseudo-class */ p::first-line { font-weight: bold; } /* pseudo-element */ nav > a { display: block; } /* child combinator */ h2 + p { margin-top: 0; } /* adjacent sibling */ [type="email"] { border: 2px solid; } /* attribute */

📦 The CSS Box Model

/* Every element is a box: content → padding → border → margin */ .box { width: 200px; /* content width */ padding: 16px; /* inside spacing */ border: 2px solid #333; margin: 24px auto; /* outside spacing */ /* box-sizing: border-box makes width include padding+border */ box-sizing: border-box; }
💡

Always use box-sizing: border-box globally. It makes layout math much more intuitive.

🌈 Colors & Units

/* Color formats */ .a { color: red; } /* keyword */ .b { color: #1d4ed8; } /* hex */ .c { color: rgb(29,78,216); } /* rgb */ .d { color: rgba(29,78,216,0.5); } /* rgba — with opacity */ .e { color: hsl(222,76%,48%); } /* hsl */ /* Units */ .f { font-size: 16px; } /* absolute pixels */ .g { font-size: 1rem; } /* relative to root font-size */ .h { font-size: 1.2em; } /* relative to parent */ .i { width: 50%; } /* % of parent width */ .j { width: 50vw; } /* 50% of viewport width */

⚖️ Specificity (who wins?)

/* Specificity score: (IDs, Classes, Elements) */ p { color: black; } /* (0,0,1) */ .highlight { color: yellow; } /* (0,1,0) — wins over p */ #hero p { color: blue; } /* (1,0,1) — wins over .highlight */ /* !important overrides everything — use sparingly */ p { color: red !important; } /* Order matters when specificity is equal — last rule wins */ p { color: blue; } p { color: green; } /* this wins */
