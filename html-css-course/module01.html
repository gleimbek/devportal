What is HTML?

HyperText Markup Language — the skeleton of every web page.

HTML at a Glance

HTML (HyperText Markup Language) tells the browser what content is on a page — headings, paragraphs, images, links, and more. It uses tags wrapped in angle brackets.

💡

HTML ≠ a programming language. It is a markup language — you describe content, not logic. For logic you add JavaScript.

The Minimal HTML Document

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first web page.</p> </body> </html>

Every HTML file should start with this structure. Let's break it down:

  • <!DOCTYPE html> — Tells the browser this is an HTML5 document.
  • <html lang="en"> — The root element; lang helps screen readers and search engines.
  • <head> — Invisible metadata: character set, title, CSS links.
  • <body> — Everything the user sees on screen.
🏷️

Tags, Elements & Attributes

The building blocks of every HTML document.

Anatomy of an HTML Tag

<!-- Opening tag + content + closing tag = element --> <p>This is a paragraph.</p> <!-- Self-closing (void) element — no closing tag needed --> <img src="photo.jpg" alt="A landscape"> <!-- Attribute format: name="value" --> <a href="https://example.com" target="_blank">Visit</a>
⚠️

Always close your tags (except void elements). Unclosed tags cause unpredictable rendering bugs.

📝 Core Text Tags

<h1>Main Heading (biggest)</h1> <h2>Sub-heading</h2> <h3>Section heading</h3> <p>A paragraph of text.</p> <span>Inline container</span> <div>Block container</div>

✨ Inline Formatting

<strong>Bold / important text</strong> <em>Italic / emphasized text</em> <mark>Highlighted text</mark> <code>Inline code</code> <br> <!-- line break (void) --> <hr> <!-- horizontal rule (void) -->

💬 HTML Comments

<!-- This is a comment — the browser ignores it --> <!-- Multi-line comment. Use comments to explain your code or temporarily disable sections. --> <p>This text shows.</p> <!-- <p>This paragraph is hidden.</p> -->
