HTML Tables

For tabular data — not for layout!

Table Structure

<table> <caption>Monthly Sales 2024</caption> <thead> <tr> <th scope="col">Month</th> <th scope="col">Revenue</th> <th scope="col">Units</th> </tr> </thead> <tbody> <tr> <td>January</td> <td>$12,400</td> <td>320</td> </tr> <!-- more rows... --> </tbody> <tfoot> <tr> <td>Total</td> <td colspan="2">$148,800</td> <!-- spans 2 cols --> </tr> </tfoot> </table>

🎨 Custom List Styles

/* Remove default bullets */ ul.clean { list-style: none; padding: 0; } /* Custom bullet with emoji */ ul.check li::before { content: "✅ "; } /* Ordered list styles */ ol.alpha { list-style-type: lower-alpha; } ol.roman { list-style-type: upper-roman; } ol.start5 { start: 5; } /* begin at 5 */ /* Horizontal nav list */ nav ul { display: flex; list-style: none; gap: 16px; }

🎨 Styling Tables with CSS

table { width: 100%; border-collapse: collapse; /* merge cell borders */ } th { background: #1d4ed8; color: #fff; padding: 10px 14px; text-align: left; } td { padding: 9px 14px; border-bottom: 1px solid #e2e8f0; } tr:nth-child(even) td { background: #f8fafc; } tr:hover td { background: #eff6ff; } /* Responsive: wrap in a scroll container */ .table-wrap { overflow-x: auto; }

🔲 Merging Cells

<!-- colspan spans across columns, rowspan spans down rows --> <table> <tr> <th colspan="3">Full Year Summary</th> </tr> <tr> <td rowspan="2">Q1 & Q2</td> <td>Jan–Mar</td> <td>$36k</td> </tr> <tr> <td>Apr–Jun</td> <td>$41k</td> </tr> </table>
