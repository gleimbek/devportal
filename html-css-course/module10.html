Form Structure & the <form> Element

Every form starts with <form> — it wraps all controls and defines how data is sent.

The <form> Element

<form action="/submit" <!-- where data is sent --> method="POST" <!-- GET (URL) or POST (body) --> novalidate <!-- disable built-in browser validation (for custom JS) --> autocomplete="on" <!-- enable browser autofill --> > <!-- Form controls go here --> </form>

GET vs POST

  • GET — appends data to URL. Use for search forms, never for passwords.
  • POST — sends data in the request body (hidden). Use for login and sensitive data.

The Label Connection — Critical for Accessibility

<!-- Method 1: for/id pairing (recommended) --> <label for="username">Username</label> <input type="text" id="username" name="username"> <!-- Method 2: wrapping (implicit association) --> <label> Password <input type="password" name="password"> </label> <!-- ❌ WRONG — input with no label --> <input type="text" placeholder="Username"> <!-- placeholder ≠ label! -->
⚠️

Placeholder is NOT a label. It disappears when typing. Always use a visible <label>.

📝 All Input Types

<!-- Text inputs --> <input type="text"> <!-- single-line text --> <input type="email"> <!-- validated email --> <input type="password"> <!-- masked text --> <input type="tel"> <!-- phone (numeric keyboard on mobile) --> <input type="url"> <!-- validated URL --> <input type="search"> <!-- search box (with clear button) --> <textarea rows="4"></textarea> <!-- multi-line text --> <!-- Numbers & ranges --> <input type="number" min="1" max="100" step="1"> <input type="range" min="0" max="100"> <!-- slider --> <!-- Date & time --> <input type="date"> <input type="time"> <input type="datetime-local"> <input type="month"> <!-- Choices --> <input type="checkbox" name="agree"> <input type="radio" name="plan" value="free"> <!-- Other --> <input type="color"> <!-- color picker --> <input type="file" accept=".pdf,.jpg"> <input type="hidden" value="csrf-token"> <input type="submit" value="Send"> <input type="reset"> <!-- clears all fields --> <button type="submit">Send</button>

✅ HTML5 Built-in Validation

<!-- Validation attributes --> <input required> <!-- must not be empty --> <input minlength="3"> <!-- min character count --> <input maxlength="50"> <!-- max character count --> <input min="18" max="99"> <!-- numeric range --> <input pattern="[A-Za-z]{2,}"> <!-- regex pattern --> /* CSS to style valid/invalid states */ input:valid { border-color: #22c55e; } input:invalid { border-color: #ef4444; } /* But only show red AFTER the user interacts */ input:invalid:not(:placeholder-shown) { border-color: #ef4444; background: #fef2f2; }
💡

Always validate on the server too. HTML5 validation is a convenience — users can bypass it with DevTools.

🗂️ Fieldset, Legend & Select

<!-- Fieldset groups related controls --> <fieldset> <legend>Shipping Address</legend> <label for="city">City</label> <input type="text" id="city" name="city"> </fieldset> <!-- Drop-down select --> <label for="country">Country</label> <select id="country" name="country" required> <option value="">— Select a country —</option> <optgroup label="North America"> <option value="mx">Mexico</option> <option value="us">United States</option> </optgroup> </select> <!-- Multi-select --> <select name="skills" multiple> <option>HTML</option> <option>CSS</option> <option>JavaScript</option> </select> <!-- Datalist: free text + suggestions --> <input list="browsers" name="browser"> <datalist id="browsers"> <option value="Chrome"> <option value="Firefox"> <option value="Safari"> </datalist>

🎨 Styling Forms with CSS

/* Reset browser defaults first */ input, textarea, select, button { font-family: inherit; font-size: 1rem; box-sizing: border-box; } /* Style the field */ input[type="text"], input[type="email"], textarea, select { display: block; width: 100%; padding: 10px 14px; border: 2px solid #e2e8f0; border-radius: 8px; outline: none; transition: border-color 0.2s, box-shadow 0.2s; } input:focus, textarea:focus, select:focus { border-color: #3b82f6; box-shadow: 0 0 0 3px rgba(59,130,246,0.2); } /* Submit button */ button[type="submit"] { background: #1d4ed8; color: #fff; border: none; border-radius: 8px; padding: 12px 28px; font-weight: 700; cursor: pointer; transition: background 0.2s; } button[type="submit"]:hover { background: #1e40af; }
