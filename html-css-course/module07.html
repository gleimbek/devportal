The <img> Element

Core attributes and best practices.

Essential Image Attributes

<!-- Minimum required: src + alt --> <img src="photo.jpg" alt="A golden retriever playing in the park" width="800" height="600" loading="lazy" > <!-- Decorative image — empty alt, browser skips it --> <img src="divider.svg" alt="" role="presentation"> <!-- Image inside a figure --> <figure> <img src="chart.png" alt="Bar chart showing Q4 revenue by region"> <figcaption>Figure 1: Q4 2024 Revenue</figcaption> </figure>
⚠️

Always include width and height attributes — they prevent layout shifts (CLS) while the image loads. The browser reserves the space before the image arrives.

📐 srcset & sizes

<!-- Browser picks the best image for the screen --> <img src="photo-800.jpg" srcset="photo-400.jpg 400w, photo-800.jpg 800w, photo-1200.jpg 1200w" sizes="(max-width: 600px) 100vw, (max-width: 900px) 50vw, 800px" alt="Responsive landscape photo" >

🖼️ Art Direction with <picture>

<picture> <!-- Modern format first --> <source srcset="photo.avif" type="image/avif"> <source srcset="photo.webp" type="image/webp"> <!-- Different crop for mobile --> <source media="(max-width: 600px)" srcset="photo-portrait.jpg" > <!-- Fallback --> <img src="photo.jpg" alt="Mountain landscape"> </picture>

🎨 CSS Image Styling

/* Always responsive */ img { max-width: 100%; height: auto; display: block; } /* Fixed-size container with cover crop */ .thumbnail { width: 300px; height: 200px; object-fit: cover; /* fill, contain, cover, none */ object-position: center top; border-radius: 8px; } /* Aspect ratio box */ .ratio-16-9 { aspect-ratio: 16 / 9; width: 100%; object-fit: cover; }

📋 Image Format Guide

Format | Best for | Support ---------|-----------------------------|--------- AVIF | Photos, high compression | Modern browsers WebP | Photos + transparency | 95%+ browsers JPEG | Photos (legacy fallback) | Universal PNG | Transparency, screenshots | Universal SVG | Icons, logos, illustrations | Universal GIF | Simple animation (use video!)| Universal Tip: Use AVIF → WebP → JPEG in <picture> Aim for < 200KB for above-the-fold images
