# 🌐 DevPortal — Web Development Learning Platform

A bilingual (English / Spanish) web development learning portal offering structured courses, interactive exercises, module quizzes, and certification exams in HTML5, CSS3, and JavaScript.

> Built as part of the **Devportal** educational ecosystem.

---

## 🚀 Live Demo

https://github.com/gleimbek/Devportal

---

## 📚 Features

- **Bilingual UI** — Full English/Spanish toggle on all pages, courses, and exams
- **Structured Courses** — HTML5 & CSS3 combined course + JavaScript course with modular lessons
- **Certification Exams** — 50-question final exams per technology (HTML, CSS, JavaScript)
- **Passing threshold** — 80% minimum score required for certification
- **Downloadable Certificates** — Auto-generated PDF certificate upon passing any final exam
- **SCORM Integration** — Compatible with LMS platforms that support SCORM
- **Responsive Design** — Works on desktop and mobile devices

## 🛠️ Tech Stack

- **HTML5** — Semantic markup and structure
- **CSS3** — Custom styling, grid layout, dark/light section themes
- **Vanilla JavaScript** — Language switching, quiz logic, score tracking
- **Canvas API** — Certificate generation
- **SCORM API** — LMS integration for progress tracking

---

## 🌍 Bilingual Support

All UI elements use `data-en` / `data-es` attributes toggled dynamically via JavaScript. This makes the platform ideal for bilingual classrooms and self-learners in both English-speaking and Spanish-speaking environments.

---

## 📜 Certification

Students who score **80% or higher** on any final exam receive a downloadable certificate of completion recognizing their achievement in that technology.

---

## 🏫 Context

This platform was developed to support students from beginner to intermediate levels.
